## 说明

本教程不是公式化,比如酱油 5ml,盐 5g 的,因为同样的酱油也有咸的和淡的,做饭不是写程序,不需要那么精确,99%的情况大概大约就足以,你也不用知道加一勺是多少,多大的勺,一勺是平的还是满的,加量这一部分你只要记住,边加边尝就完事了,一切都会刚刚好。只要你先尝试一次,把代码（bushi）把菜做出来,然后之后慢慢修复 bug 就好了。
